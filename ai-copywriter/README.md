# AI Copywriter - 智能文案工厂

✨ AI 驱动的爆款文案生成器，一键生成多平台爆款标题、文案、标签

## 功能特性

- 🎯 爆款标题生成器 - 基于数据科学，生成高点击率标题
- ✍️ 多平台文案适配 - 小红书、抖音、微博、公众号自动适配
- 🔥 热门标签推荐 - 实时抓取热门话题标签
- 📊 文案效果预测 - AI 预测文案传播潜力
- 🎨 情感风格调节 - 正式/幽默/犀利/治愈多风格切换
- ⚡ 批量生成 - 一次输入，生成 10 个版本供选择

```
帮我生成 5 个关于"AI工具"的爆款标题
```

### 多平台文案
```
写一篇关于新手机发布的小红书文案
```

### 标签推荐
```
推荐一些科技类热门标签
```
