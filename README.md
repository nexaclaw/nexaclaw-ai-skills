## 📦 技能列表

### 1. 🔥 Trending Tracker - 热点追踪器

实时追踪全网热点，抢占流量先机

**功能特性：**
- ✅ 微博热搜实时抓取
- ✅ 知乎热榜监控
- ✅ B站热门追踪
- ✅ 抖音热点分析
- ✅ AI 趋势分析
- ✅ 自定义关键词监控
- ✅ 定时推送报告

**安装：**
```bash
clawhub install nexaclaw/trending-tracker
```

**使用：**
```
帮我看看今天微博和知乎有什么热点
监控关键词 "AI", "大模型"
每天早上9点推送热点报告
```

**目录：** [trending-tracker/](./trending-tracker/)

安装后，直接在 AI Agent 对话中使用自然语言触发：
