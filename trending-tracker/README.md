# Trending Tracker - 热点追踪器

🔥 AI 驱动的全网热点追踪工具，助你抢占流量先机

## 功能特性

- 📊 实时追踪微博、知乎、B站、抖音等平台热点
- 🤖 AI 分析热点趋势和话题热度
- ⏰ 定时推送热点报告到指定频道
- 🔔 自定义关键词监控和提醒
- 📈 热点数据可视化导出
